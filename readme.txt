Selamat datang di channel Programing Mojokerto

Tutorial Membuat Face recognition dari sebuah gambar
dengan menggunakan dataset custom/ buat sendiri :)

oke kita mulai !

1. Buka channel youtube Programing Mojokerto dan lihat channel
yang kalian ingin pelajari. setelah itu download source code program
caranya : like dan komen dahulu :)
lalu klik link yang tertera pada deskripsi dan download !

2. ekstrak dan buat folder dengan nama "dataset"

3. buka folder dataset dan buat 2 folder lagi.
- folder 1 beri nama kamu : sebagai dataset identification kamu
- folder 2 beri nama unknown : sebagai pembanding dalam komputasi

4. buat dataset custom kamu sendiri dengan menambahkan foto kamu
- beri ukuran 50px x 50px atau terserah kamu asal simetris persegi
- masukkan ke dalam folder yang berisi nama kamu tadi
- masukkan juga image untuk folder unknown, usahakan wajah indo :)
- bisa capture dari ig, fb, dsb. lalu letakkan pada folder unknown

5. buat folder "images" dan masukkan foto kamu yang ingin di recognize

6. running program. buka cmd yang sudah terinstall python
dan opencv.
- masuk ke directory lokasi program dengan syntax "cd (lokasinya)"
bisa di copas aja.
- masuk ke environment python "activate (nama env)"

7. running program sesuai urutan berikut:
- buat dataset : untuk membuat datasetnya. kamu bisa copas skript
running yang sudah tersedia didalam program
- pelatihan dataset : proses pembelajaran program untuk recognize
- recognize gambar : proses untuk face recognition gambar dari foto 
yang kamu letakkan di folder images tadi. ganti dengan nama foto yang ingin di recognize, coba dengan foto lain.

*note : anda bisa menambahkan subjek recognition dengan menambahkan datasetnya. klik "space" untuk close program.

silahkan SUBSCRIBT bila anda telah berhasil !!

thanks :)
